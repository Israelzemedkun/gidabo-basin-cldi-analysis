Machine learning-based land degradation assessment for the Gidabo Basin, Ethiopia.

# Land Degradation Monitor (ELD-Monitor)

### Project Overview
This project provides a data-driven approach to monitoring land degradation within the **Gidabo River Basin** and **Tana-Beles** watersheds between 2000 and 2025. By integrating multi-source satellite imagery (Landsat, MODIS, and Sentinel) with climate and topographic data, the project develops a **Combined Land Degradation Index (CLDI)**. 

### Key Features
* **Multi-Index Analysis:** Tracks vegetation health (NDVI), soil salinity (SI), and soil exposure (BSI).
* **Interactive Dashboard:** A web-based interface for spatio-temporal trend visualization.

### Repository Structure
- `/scripts`: Python and GEE scripts for data processing.
- `/data`: Geographic boundaries and tabular datasets.
- `/app`: Files for the Streamlit dashboard.
