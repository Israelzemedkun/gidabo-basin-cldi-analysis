# Gidabo Basin Dashboard

Interactive Streamlit dashboard for the CLDI land degradation analysis.

## Run

From the **project root**:

```bash
streamlit run app/streamlit_app.py
```

Then open http://localhost:8501 in your browser.

## Requirements

Install dependencies first (if not already done):

```bash
pip install -r requirements.txt
```

## Features

- Interactive scatter map (OpenStreetMap) with basin boundary overlay
- Filter by Zone and Degradation Status
- Live metric cards, CLDI histogram, and zone breakdown chart
